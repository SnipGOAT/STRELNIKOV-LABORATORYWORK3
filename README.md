# *15. Итераторы и генераторы*
*1) Реализуйте класс-итератор для последовательности Фибоначчи с заданным максимальным значением.*

*2) Реализуйте генератор для последовательности простых чисел до заданного границы.*

*3) Реализуйте класс-контейнер для коллекции книг, который поддерживает перебор (итерацию) по своим элементам.*

# 1. Суть задания
Необходимо разработать программу на языке Python, которая демонстрирует три различных подхода к работе с итерациями:
- Класс-итератор для генерации чисел Фибоначчи до заданного максимального значения.
- Функция-генератор для получения последовательности простых чисел до заданной границы.
- Класс-контейнер для коллекции книг, поддерживающий протокол итерации (перебор элементов в цикле `for`).
# 2. Описание требований
Особенность реализации: задание требует понимания разницы между итераторами (классы с методами `__iter__` и `__next__`) и генераторами (функции с ключевым словом `yield`). Также необходимо реализовать собственный контейнерный класс, который делает коллекцию объектов итерируемой.
# 3. Алгоритм решения
## Шаг 1: Класс-итератор Фибоначчи
Создаётся класс `FibonacciIterator` с конструктором, принимающим максимальное значение.
Метод `__iter__` возвращает сам объект итератора.
Метод `__next__` вычисляет следующее число Фибоначчи. Если значение превышает лимит, выбрасывается исключение `StopIteration`.
## Шаг 2: Генератор простых чисел
Создаётся функция `prime_generator` с параметром `limit`.
Вспомогательная функция `is_prime` проверяет число на простоту (делится только на 1 и само себя).
Цикл перебирает числа от 2 до `limit`. Простые числа отдаются через `yield`.
## Шаг 3: Класс-контейнер для книг
Создаётся класс `Book` для представления отдельной книги (название, автор).
Создаётся класс `BookCollection`, который хранит список книг.
Реализуется метод `__iter__`, который возвращает итератор по внутреннему списку книг, позволяя использовать коллекцию в цикле `for`.
# 4. Ожидаемые результаты
Программа выводит результаты в консоль:
- Последовательность чисел Фибоначчи, не превышающих заданный лимит.
- Список всех простых чисел в заданном диапазоне.
- Демонстрация перебора коллекции книг в цикле с выводом информации о каждой книге.
# 5. Вывод
